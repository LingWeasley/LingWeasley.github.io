远程服务器链接

发表于2023-01-12更新于2023-06-21
  1. 通过ipconfig/ifcofig查看服务器的IP地址，如果没有反映可以sudo apt install net-too (是一个简单的网络管理工具，为了查看IP)，查看到IP地址以后可以在本机断ping一下IP地址，看一下是否能ping通
  2. 通过cat/etc/passwd 是机器的密码本，可以查看机器有什么用户
  3. pwd–查看当前用户
  4. code / etc/ssh/ssh_config 查看ssh配置
  5. sudo systemctl restart sshd —systemctl是系统管理工具，restart是命令。sshd 是 d是ssh的守护程序？
  6. 装一个ssh-server –sudo apt install openssh-server ssh开源的服务器端
  7. sudo systemctl status ssh 检查有没有ssh-server
  8. ssh ubuntu@10.5.2.6 在本机上 链接
  9. 下一步就是配置vscode或者pycharm
  10. 在vs code上要下载一个remote-ssh的插件
  11. 通过command+shifte+p 打开命令面板，输入ssh，增加一个新的ssh hoste,输入ssh ubuntu@10.5.2.6,然后把这个ssh ubuntu@10.5.2.6配置保存到（用户或者系统）目前保存到用户上，在访达的用户中要找隐藏文件，通过command +. 查看隐藏文件，打开隐藏文件可以看到保存的hoste ,私钥和公钥，如果密钥出现问题可以重新新建
  12. 新建密钥，ssh-keygen –之后是生成的位置，另外的一个短语不需要就直接回车
  13. 在本机终端通过ssh 服务器名- IP地址进行连接。/或者可以给服务器起一个别名
  14. 需要输入服务器密码
  15. 如果要免密登陆的话需要在本机上传密钥（pud） –ssh-copy-id ubuntu@10.5.2.6 把密钥上传到ubuntu@10.5.2.6 这个服务器上
  16. 可以通过htop 查看服务器的运行情况
  17. 在vscode上有一个插件，audio preview可以用来听wav文件并进行分析
